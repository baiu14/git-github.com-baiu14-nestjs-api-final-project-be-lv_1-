## Deploy To Server
1. Install Nginx
2. Adding Project to the Server
3. Installing Node.js and Dependencies
4. Configure .ENV file (APP_PORT as 3000, APP_NAME and DATABASE_URL )
5. Running Project on Server (on port 3000)
6. Configuring NGINX as a Reverse Proxy :
  6.a. Create /etc/nginx/sites-available/"PROJECT_NAME"
  6.b. FIll file with code :
```
  server {
    listen 80;
    listen [::]:80;

    root /var/www/html;
    index index.html index.htm index.nginx-debian.html;

    server_name your-domain.com;

    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
  }
```
  6.c. Enable site "PROJECT_NAME"
  ```
  sudo ln -s /etc/nginx/sites-available/"PROJECT_NAME" /etc/nginx/sites-enabled
  ```
7. Setting Up Firewall:
  7.a. Enable the firewall
  ```
  sudo ufw enable
  ```
  7.b. Allow port HTTP and HTTPS
```
  sudo ufw allow 80
  sudo ufw allow 443
```
8. Restart Nginx
